# MSR Command Center Frontend

Production-style Next.js App Router frontend for the MSR Command Center SaaS demo.

## Run

```bash
npm install
npm run dev
```

## Verify

```bash
npm run lint
npm run build
```

## What’s Included

- Marketing pages for the landing and pricing experience.
- Auth pages for login, registration, and password recovery.
- Authenticated app shell with sidebar, top bar, command palette, and toast provider.
- Personal and manager workflows for boards, MSRs, projects, cycles, reports, and admin views.
- Seeded mock data and a mock API layer so the app behaves like a product instead of a placeholder.
